# Projeto 2.2: Criador de Histórias

`Objetivo:` Criar um programa interativo que constrói uma pequena história usando palavras
fornecidas pelo utilizador.

`Requisitos:`
1. Use input() para pedir ao utilizador várias palavras (ex: um nome, um verbo, um
adjetivo, um lugar).
2. Armazene cada palavra numa variável com um nome descritivo.
3. Use múltiplas instruções print() para montar e exibir uma história curta e divertida.
4. Experimente com os argumentos sep (separador) e end da função print() para formatar
a saída da história.
